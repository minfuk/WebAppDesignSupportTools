I want to build the tool supporting web application developments.

I focus to the applicaitons that have the following features.

1. They can be requested in the x-www-form-urlencoded format and response in the HTML format.
2. They use the relational database for the background data store.


The design tool can do following things.

1. generate the design documents.
2. generate the test data for the application.
3. generate the solution of the given constraints.
